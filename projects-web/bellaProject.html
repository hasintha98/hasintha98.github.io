Bella fashion online store is a full-stack MERN web application using redux technology to manage the states of the system. It has consisted of many functions according to the admin side and the client-side of the system.

User Interfaces

User Management

This is all about the users that register to our system. As an admin, he is able to manage the users in the system. admin can add users manually and create roles for users according to the permissions that the user wants. Not only that, the admin can delete the registered user from the system.

Catagory Management

This is all about the categories of the ongoing products in our system. As an admin, he will be able to manage the categories in the system. admin can add/edit/delete categories of the system.

Product Management

This is all about the products of our system. The product can be different types such as shoes, top wear, and bottom wear. As an admin, he will be able to manage the products in the system. admin can add/edit/delete products of the system. Admin can also add brands to the system in this management section. after adding the brand, the admin can assign those created brands to products when the admin creates a product.

Order Management

This is all about the Orders placed by clients of our system. As an admin, he will be able to manage the orders in the system. order status can be seen in the admin section. admin can add/edit/delete ongoing orders of the system.
