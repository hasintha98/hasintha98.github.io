Description - Mobile application

The Nation of Gamers mobile app users can conveniently explore the Latest news, Fun and interesting gameplay, live streams, and newly released Game trailers. Because of The user-friendly interface anyone can figure out, that they have what they want as a gamer. There's something for everyone. Hottest gaming industry news, including the latest and greatest game news, highly anticipated releases, and recent updates of their favorite games. Admin uses can provide new/streams of every related aspect of gaming industry. From big-budget Triple-A games that came out on PC, PlayStation, and Xbox to up and rising developer's Indie games that users might be interested. Finally, users can access the nation of gamers YouTube channel by using the mobile app. Mobile app is already published on google play store. Click here.

Description - Admin web application

This admin console can create, edit delete any news to the system. Not only that, but they can also create a stream (live event) with a given time period to go live on the mobile app. When creating events such that, it will generate a notification and send all the mobile phones where the app has installed. In the admin management, there is a point system for every added news and streams. They can score points by adding each news and streams to the system. According to the points, system will generate a rank and display on the admin console dashboard. Admin will be able to motivate to create news and streams by reason of scoring their points. In the YT management, all videos are fetched from the nation of gamers YouTube channel. With the help of the notification button that has on every column we can send a notification to all devices regarding the video whenever you want. Finally, in the homepage management, an admin can configure the home page title, subtitle and the wallpaper that needs to be display in the mobile app.
